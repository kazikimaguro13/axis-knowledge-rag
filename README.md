# axis-knowledge-rag

YAML frontmatter 付き Markdown ナレッジに対する、軸検索 + RAG 検索のローカル Web アプリ OSS。

## ステータス

🚧 開発中 (v0.1.0 リリース予定: 2026-05-18)

## 特徴 (3 週間で実装予定)

- **軸メタデータでの構造化検索 + ベクトル検索のハイブリッド**
- 日本語ナレッジ特化 (表記ゆれ吸収予定)
- LangChain / LlamaIndex 不使用、自前実装
- Local-first 設計 (個人データを外部送信しない)
